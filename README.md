### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/sasabakic/ld-biro-fe.git
   cd ld-biro-fe
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install

   ```

   Navigate to [http://localhost:3000](http://localhost:3000)
